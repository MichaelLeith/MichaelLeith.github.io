Weight

A puzzle platform game prototype created for the 48 hour Ludum Dare game jam. The goal is to reach an orb at the end of each level. To do this the player has to "eat" orbs, which cause them to gain weight, and allow them to break through coloured tiles in the map. They have to move quickly though, as they slowly lose weight overtime.

The game was written from scratch using the HTML5 canvas, and the source can be found here.

Dungeon

A short 3D game created for my university computer graphics course. The player must solve puzzles to gain the ability to scare the dungeon's guards, and get the treasure!

The game was written using OpenGL 3.1 and C++ (with SFML for audio). All modelling was done in Blender.

The game's source can be found here.
